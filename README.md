# CabinClear UK — Car Interior Organisation Store

Premium car interior organisation e-commerce storefront built with React 19 + TypeScript + Vite + Tailwind CSS.

## Deploy to Netlify

1. Run `npm install`
2. Run `npm run build`
3. Drag the `dist/` folder to [netlify.com/drop](https://app.netlify.com/drop)

## Tech Stack
- React 19 + TypeScript
- Vite
- Tailwind CSS 3 + shadcn/ui
- Lucide React icons

## Currency
All prices in GBP (£) — UK market pricing.
